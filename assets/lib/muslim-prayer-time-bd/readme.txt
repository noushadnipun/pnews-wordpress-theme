"Muslim Prayer Time BD" plugin provides the ability to display prayer (salat) times for BD Muslims with pretty widget.


== Description == 
"Muslim Prayer Time BD" plugin provides the ability to display prayer (salat) times for BD Muslims with pretty widget. The plugin has been developed using the perpetual prayer calendar of Bangladesh. Now you can remind you and your blog's visitors about the importance of salat or praying to Allah.

From the frontend, you can display a daily prayer timetable for your own place. In the month of Ramadan you can also display Sehri last time by enabling Sehri Time which is located under Settings screen in the dashboard. There you can also change the background color, font color, font weight and text align of the table.


= Usage =
* Install and activate the plugin. Go to your Dashboard, then navigate to "Appearance >> Widgets". There you will get 'T4B Muslim Prayer Time' widget. Just drag and drop the widget in your sidebar. Visit your site and you will see the Prayer Timetable in your sidebar.


= Features: =
- Auto generated prayer timetable.
- Option to display prayer timetable for any district.
- Option to select the default city.
- Option to display sehri last time.
- Unlimited background color for prayer name and time.
- Unlimited background color for sehri time.
- Unlimited font color for prayer name and time.
- Unlimited font color for sehri time.
- Option to set font weight.
- Option to set text align.

== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Install and activate the plugin.
3. Go to: Appearance >> Widgets.
4. Drag and drop 'T4B Muslim Prayer Time' widget in your sidebar.
5. To display Sehri last time go to Settings >> Prayer Settings and enable Sehri time.
